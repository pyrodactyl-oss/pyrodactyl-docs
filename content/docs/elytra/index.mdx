<Callout type="error">
  Elytra is **no longer being updated**. It was a vibe-coded fork of Wings by PyroHost that ended up being worse than the original.
  We recommend using upstream [Pterodactyl Wings](https://pterodactyl.io/wings/1.0/installing.html) instead.
</Callout>

<Callout type="info">
    Elytra is a daemon that runs on game servers and communicates with your Pyrodactyl Panel to manage server instances.
</Callout>
