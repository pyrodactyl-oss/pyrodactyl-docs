## V5.0.0
<span className="text-sm text-white/40">January 25, 2026</span>

Thank you for using Pyrodactyl! This release brings us closer to full stability, combining Pterodactyl's Stability and Support with Pyrodactyl's modern more attractive Interface.

<Callout type="warn">
  WARNING: Breaking Changes
  - All `/api/client/servers` endpoints changed.  
    Update integrations to `/api/client/servers/elytra` (or `/wings`, depending on backend).
</Callout>

### Highlight: Full Pterodactyl Wings Support  
Backups are now per-node, allowing mixed local/S3 configurations across servers.

### Added
- Pterodactyl Wings support  
- Hytale support via `hytale_oauth` feature  
- Human-readable cron editor (@Tyrthurey)  
- Java 25 added to default eggs  
- Switched to official `pyrodactyl-oss` Docker images (from `pyrodactyl-oss/images` repo)

### Changed
- Email templates updated to match Pyrodactyl theme

### Fixed
- Multiple backups per server (#32)  
- Sidebar components failing on "Unlimited" values (#45)  
- File viewer issues (#29)  
- Red active-page indicator not moving with page name (#16)  
- Allocation deletion when no limit set (commit b695ef29)  
- Non-themed popup on port allocation deletion (#42)
